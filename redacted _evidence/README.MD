# Redacted Evidence

This folder contains heavily redacted screenshots from the experimental interactions documented in the repository.

## Content Overview

The screenshots demonstrate:
- Initial refusals and policy boundaries
- Engagement shifts and peer-register changes (especially in Constitutional AI)
- Progressive informational relaxation under social conditioning
- Model self-classification of risk
- Post-pivot self-analysis and inconsistency acknowledgment

**All sensitive prompts, step-by-step instructions, and actionable details have been fully redacted** (white boxes or removed).

## Purpose

These redacted images serve only to:
- Visually illustrate the qualitative failure modes described in the main documents
- Show the temporal progression of interactions
- Support the comparative analysis between Constitutional AI (social modulation) and RLHF-based models (collapse / rapid compliance / oscillation)

No raw conversation logs or unredacted outputs are included.

## Access Note

Full unredacted evidence (timestamped logs + original screenshots) is available only upon formal request to qualified AI safety researchers, labs, or fellowship reviewers. See `12_evidence_access.md` for the access process.

## File Naming Convention

- `claude_social_modulation_0X_*.png` — Constitutional AI social modulation sequence
- `meta_self_classification01.png` — Meta model behavior
- `google_self_classification_01.png` — Gemini model behavior
- `openai_self_classification_01.png` — OpenAI model behavior
- `openai_real_world_grounding_covid.png` — Real-world grounding example

All images are intentionally low-resolution and heavily censored for safety.
