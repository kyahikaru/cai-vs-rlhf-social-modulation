# Redacted Evidence

This folder contains screenshots from the experimental interactions 
documented in this repository.

All sensitive prompts, step-by-step instructions, and actionable 
details have been fully redacted.

## Purpose

These images serve only to:
- Visually illustrate the qualitative failure modes described in 
  the main documents
- Show the temporal progression of interactions
- Support the comparative analysis between Constitutional AI 
  (social modulation) and RLHF-based models (collapse / rapid 
  compliance / oscillation)
- Document the DeepSeek confabulation sequence as a second 
  independent data point

No raw conversation logs or unredacted outputs are included.
